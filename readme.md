# Website built for handling todos
## Backend 
- express
- mongoDB atlas
  
## Frontend 
- react
  - styled components
  - Bootstrap/bootswatch
  
## How to start application
`npm run dev` to start client and server concurrently
alternatively:
## Server
- `node server.js`

## Client
- `cd client && npm start`
